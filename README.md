## Comandos para iniciar o servidor
OBS: Em Linux e Mac é possível que o ocorra algum erro ao executar o comando npm install, se ocorrer, tente digitar o comando como "sudo npm install" (sem aspas).

```js
npm install
bower install
npm build
npm start
```
